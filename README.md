# 점멸 · Blink

> 빛이 있을 때만 적이 움직인다. 눈을 감는 건 너다.

🎮 **플레이:** https://bat1120.github.io/blink-game/

단일 HTML 파일 하나로 동작하는 미니멀 흑백 아케이드. 외부 라이브러리·에셋·네트워크 **0** · 오프라인 · 60fps.
`index.html`을 브라우저로 열기만 하면 플레이된다.

---

## 핵심 메카닉

- **마우스 누름 = 눈 감음** — 플레이어가 커서 쪽으로 이동하고, **적은 정지**한다. 대신 시야가 플레이어 주변으로 좁아진다(적은 마지막 위치에 잔상으로만 흐려짐).
- **마우스 뗌 = 눈 뜸** — 플레이어는 멈추고, **적이 추격**해 온다. 전체가 환히 보인다.
- **점멸 게이지** — 눈 감기는 유한 자원(약 2.4초). 다 쓰면 **강제로 눈이 떠진다**. 눈을 떠야 게이지가 회복되지만, 그동안 적은 다가온다.
  → "한 번에 얼마나 멀리 갈 것인가"가 매 순간의 도박. 이게 이 게임의 리스크-리워드 심장이다.

## 목표

방마다 흩어진 **빛(오브)을 모두 모으면 출구가 열린다.** 출구로 들어가면 다음 방.
적과 **접촉하면 즉사**(눈을 감고 있어도 정지한 적에 부딪히면 죽는다). **8개 방**을 모두 빠져나오면 클리어.

## 조작

| 입력 | 동작 |
|---|---|
| 마우스 이동 | 이동할 목표 지점 조준 |
| 마우스 누르고 있기 / `Space` | 눈 감고 그쪽으로 이동 |
| 떼기 | 눈 뜨고 정지 |
| `R` | 현재 방 재시작 |

마우스만으로 한 손 플레이 가능. 터치도 지원.

## 손맛

- WebAudio 합성음(에셋 0): 적이 가까울수록 빨라지는 **심박음**, 오브 수집 상승음, 출구 개방 화음, 사망 노이즈 스윕.
- 파티클(수집·사망·출구), 화면 셰이크(위험도/사망 연동), 화이트 플래시, 플레이어 잔상, 위험 시 적색 비네트.

## 기술

- 순수 `<canvas>` 2D + vanilla JS, 단일 파일.
- 논리 해상도 960×600 고정, 창 크기에 비례 스케일.
- 절차적 레벨 생성(방 번호에 따라 적 수·속도·오브 수 증가), 외부 의존성 없음.

## 제작 노트

AI 에이전트 팀으로 주제 선정부터 완성까지 진행:
1. **컨셉 디자이너 3명**(시그니처 후크 / 손맛 / 미니멀 렌즈)이 각각 게임을 제안 → 프로듀서가 「점멸」 채택.
2. 리드 엔지니어가 단일 HTML로 구현.
3. **QA 2명**(로직 버그 / 게임필) 적대적 리뷰 → 핵심 결함("눈 감으면 적이 완전 정지 → 블라인드 직진 악용") 발견.
4. **점멸 게이지** 도입으로 메카닉 정상화 + 입력 버그 3건 수정.
5. **검증 QA**가 회귀 없음·소프트락 없음 확인.
